# Activity Finder React

React/Vite version of the Activity Finder app.

## Run locally

```bash
npm install
npm run dev
```

Open the localhost URL shown by Vite, usually http://127.0.0.1:5173/.

## Build

```bash
npm run build
```

Logo: `public/assets/images/logo-transparent.svg`
Favicon: `public/favicon-512.png`
Data: `public/data/activities.json`
